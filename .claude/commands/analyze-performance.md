# 性能分析

## 功能
分析系统性能和瓶颈，进行性能基准测试、瓶颈识别分析、优化建议和监控配置指导。

## 参数
- endpoint: 可选，指定要分析的API端点，默认分析所有端点

## 性能分析项目
1. **API端点性能**
   - 响应时间测试
   - 并发处理能力
   - 内存使用分析

2. **数据库性能**
   - 查询执行时间
   - 连接池效率
   - 索引优化建议

3. **系统资源监控**
   - CPU使用率
   - 内存占用
   - 网络I/O

4. **优化建议**
   - 代码层面优化
   - 配置参数调整
   - 架构改进方案

5. **监控配置**
   - 性能指标设置
   - 告警规则配置
   - 日志分析建议
