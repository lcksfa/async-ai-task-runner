# 文档生成命令

## 用途
自动生成各类技术文档和学习资料

## 命令定义

### `/docs-api`
生成API文档

**功能**:
- 提取OpenAPI规范
- 生成接口使用示例
- 创建API测试脚本
- 生成文档网站
- 创建API变更日志

### `/docs-architecture`
生成架构文档

**功能**:
- 系统架构图生成
- 组件交互关系说明
- 数据流图绘制
- 部署架构文档
- 技术决策记录(ADR)

### `/docs-development`
生成开发文档

**功能**:
- 开发环境搭建指南
- 代码贡献指南
- 编码规范文档
- 测试策略说明
- 发布流程文档

### `/docs-learning [day]`
生成学习文档

**参数**:
- `day`: 可选，指定开发日期 (1-5)

**功能**:
- 每日开发总结
- 技术知识点整理
- 问题解决方案记录
- 最佳实践总结
- 学习路径规划

### `/docs-update [section]`
更新指定类型的文档

**参数**:
- `section`: 文档部分 (api, architecture, development, learning)

**功能**:
- 检测代码变更
- 自动更新相关文档
- 验证文档准确性
- 生成变更报告
- 文档版本管理
