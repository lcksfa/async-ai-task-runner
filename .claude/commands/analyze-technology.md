# 技术分析

## 功能
分析特定技术在项目中的使用情况和原理，提供技术原理深度讲解、使用场景分析、配置优化建议和学习资源推荐。

## 参数
- tech: 技术名称 (fastapi, pydantic, sqlalchemy, celery, redis, docker, mcp)

## 技术覆盖范围
1. **FastAPI**
   - 异步Web框架原理
   - 依赖注入机制
   - API文档自动生成

2. **Pydantic**
   - 数据验证机制
   - 类型系统设计
   - 序列化/反序列化

3. **SQLAlchemy**
   - ORM映射原理
   - 异步数据库操作
   - 查询优化策略

4. **Celery + Redis**
   - 分布式任务队列
   - 消息传递机制
   - 任务状态管理

5. **Docker**
   - 容器化原理
   - 服务编排
   - 环境隔离

6. **MCP (Model Context Protocol)**
   - AI协议标准
   - 工具和资源管理
   - 提示词工程
