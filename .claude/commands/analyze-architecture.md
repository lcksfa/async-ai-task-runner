# 架构分析

## 功能
深度分析项目架构和特定组件，解析组件间的交互关系，识别设计模式和最佳实践。

## 参数
- component: 可选，指定要分析的组件 (api, database, worker, mcp, all)

## 分析内容
1. **整体架构分析**
   - 分层架构设计
   - 组件间通信模式
   - 数据流向分析

2. **组件深度分析**
   - API层: FastAPI路由设计和依赖注入
   - 数据层: SQLAlchemy模型和数据库设计
   - 任务层: Celery工作者和队列管理
   - 协议层: MCP服务器实现

3. **设计模式识别**
   - 依赖注入模式
   - 异步处理模式
   - 配置管理模式

4. **最佳实践建议**
   - 代码组织优化
   - 性能改进建议
   - 安全性强化方案
